Neubrutalist Components

A collection of Tailwind CSS components in the neubrutalist style, organized using atomic design principles.

Atomic Design Structure

Atoms

Basic building blocks like buttons, inputs, and tags that serve as the foundation for all components.

Molecules

Combinations of atoms that work together as a unit, such as form groups, cards, and navigation elements.

Organisms

Complex UI components composed of molecules and atoms that form distinct sections of the interface.

Installation

Getting Started

Add these components to your Tailwind CSS project:

/* Add to your CSS */
@layer components {
  .neubrutalist-button {
    @apply px-6 py-3 font-bold text-lg transition-all duration-200 border-4 border-black bg-white;
    box-shadow: 8px 8px 0 #171717;
    transform: translate(-4px, -4px);
  }

  .neubrutalist-button:hover {
    transform: translate(0, 0);
    box-shadow: 0 0 0 #171717;
  }

  .neubrutalist-button.primary {
    @apply bg-emerald-600 text-white;
  }

  .neubrutalist-button.secondary {
    @apply bg-emerald-200 text-black;
  }
}
